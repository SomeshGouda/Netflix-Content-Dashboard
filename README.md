# 🎬 Netflix Content Analytics Dashboard

An interactive Power BI dashboard built using the Netflix Movies & TV Shows dataset to analyze content distribution, ratings, release trends, countries, and genres.

---

## 📊 Dashboard Preview

![Dashboard Preview](Netflix_Dashboard.png)

---

## 🚀 Features

- Total Titles KPI
- Movies vs TV Shows Distribution
- Top 10 Countries by Content
- Rating Distribution
- Genre Analysis
- Release Year Trend
- Content Added by Year
- Interactive Slicers
  - Type
  - Country
  - Rating
  - Release Year

---

## 🛠 Tools Used

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel
- Jupyter Notebook (Python)
